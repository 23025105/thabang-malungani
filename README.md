# Malungani Thabang Portfolio Website

 1. Project Title & Description

*Malungani Thabang Portfolio Website*

This is a personal portfolio website that showcases my skills, projects, and contact details. It introduces who I am as a developer and highlights my experience and work in a clean and modern interface.



 2. Purpose of the Website

The purpose of this website is to:

* Present my profile as a Computer Science student
* Showcase my technical skills and projects
* Provide a way for people to contact me

It solves the problem of having a centralized place where my work and information can be easily accessed.

---

 3. Features

The website includes the following features:

* *Navigation Bar*

  * Fixed header with links to different sections (Home, Skills, Projects, Contact)

* *Hero Section*

  * Introduction with profile image and personal description

* *Skills Section*

  * Animated progress bars showing proficiency in:

    * HTML & CSS
    * JavaScript
    * Python
    * React

* *Projects Section*

  * Displays project cards with:

    * Project title
    * Description of each project

* *Contact Form*

  * Users can enter:

    * Name
    * Email
    * Message

* *Responsive Design*

  * Works on both desktop and mobile devices

---

 4. Technologies Used

The following technologies were used to build this project:

 *HTML* – Structure of the website
 *CSS* – Styling, layout, and animations
 *JavaScript* – Interactive elements (skill bar animations)
*Google Fonts (Poppins)* – Typography

 5. How to Run the Project
